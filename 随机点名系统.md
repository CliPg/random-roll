# README

## 接口

| **模块** | **接口名称**     | **请求方式** | **路径**             | **主要功能**                       | 完成情况 |
| -------- | ---------------- | ------------ | -------------------- | ---------------------------------- | -------- |
| 用户管理 | 导入学生名单     | POST         | /students/import     | 从 Excel 导入学生信息到数据库      |          |
| 用户管理 | 获取学生列表     | GET          | /students/list       | 获取所有学生信息                   |          |
| 用户管理 | 获取单个学生信息 | GET          | /students/one        | 获取单个学生信息                   |          |
| 用户管理 | 导出积分表       | GET          | /students/export     | 导出当前积分表为 Excel             |          |
| 点名功能 | 随机点名         | GET          | /rollcall/random     | 按概率（积分越高概率越低）随机点名 |          |
| 点名功能 | 顺序点名         | GET          | /rollcall/sequential | 按学号顺序点名                     |          |
| 点名功能 | 提交点名结果     | POST         | /rollcall/result     | 更新学生到课与答题后的积分变化     |          |
| 积分管理 | 获取积分排行榜   | GET          | /points/rank         | 查询积分前 N 名学生                |          |
